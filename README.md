 # Visual API Orchestrator

A developer-first platform that allows you to visually design databases and instantly generate production-ready backend APIs.

##  Overview

Visual API Orchestrator eliminates repetitive backend setup by allowing developers to:

* Create database tables visually
* Automatically generate Prisma schema
* Generate fully functional CRUD APIs
* Sync database instantly
* Export a complete backend project (ready to run)

##  Problem

Setting up backend infrastructure is repetitive and time-consuming:

* Writing Prisma schemas manually
* Building CRUD endpoints for every model
* Configuring project structure repeatedly

##  Solution

Visual API Orchestrator automates the entire workflow:

UI → Schema → Database → API → Exportable Backend

All in one place.

##  Features

* Project-based workspace (save & reload)
* Multi-table database builder
* Field-level editing (type, required, updates)
* Automatic Prisma schema generation & sync
* Full CRUD API generation with validation
* Backend export (ZIP) with:

  * Express server
  * Prisma setup
  * Ready-to-run configuration
* Real-time feedback system (loading, success, error)

##  Tech Stack

Frontend:

* React + TypeScript (Vite)

Backend:

* Node.js + Express
* Prisma ORM
* SQLite

##  How It Works

1. Create a project
2. Add tables and fields
3. Edit or delete anytime
4. System generates schema and APIs
5. Export backend as ZIP
6. Run locally in seconds

##  Getting Started

```bash
npm install
npm run dev
```

Backend:

```bash
cd backend
npm install
npm run dev
```
