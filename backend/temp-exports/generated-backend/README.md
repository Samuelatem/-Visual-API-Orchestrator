## Quick Start

### 1. Install Dependencies

```bash
npm install
```

### 2. Setup Database

```bash
npx prisma db push
```

### 3. Start Development Server

```bash
npm run dev
```

The server will run on `http://localhost:5000`

## API Endpoints

Each table has the following CRUD endpoints:

```
POST   /api/<table>        Create a new record
GET    /api/<table>        Get all records
GET    /api/<table>/:id    Get a single record by ID
PUT    /api/<table>/:id    Update a record
DELETE /api/<table>/:id    Delete a record
```

## Example Usage

### Create a Record

```bash
curl -X POST http://localhost:5000/api/products \
  -H "Content-Type: application/json" \
  -d '{ "name": "iPhone", "price": 999 }'
```

### Get All Records

```bash
curl http://localhost:5000/api/products
```

### Get Single Record

```bash
curl http://localhost:5000/api/products/123
```

### Update a Record

```bash
curl -X PUT http://localhost:5000/api/products/123 \
  -H "Content-Type: application/json" \
  -d '{ "price": 1099 }'
```

### Delete a Record

```bash
curl -X DELETE http://localhost:5000/api/products/123
```
